# Duong Tran Portfolio

Personal portfolio website for Duong Tran.

## Structure

- `index.html` - main portfolio page
- `assets/images/profile.png` - profile image
- `assets/docs/duong-tran-cv.pdf` - CV download

## Local preview

Open `index.html` in a browser.

## Publish with GitHub Pages

This repository is configured to publish from the `main` branch root.

After pushing updates:
1. Go to GitHub repository settings
2. Open **Pages**
3. Confirm source is **Deploy from branch**
4. Select `main` and `/ (root)`

## Update later

Edit `index.html`, replace assets if needed, then commit and push changes to `main`.
